# User Management App - Backend

Backend server for the User Management Application built using Node.js, Express.js, and MongoDB.

---

## 🚀 Features

- User Authentication
- JWT Authorization
- Password Hashing using bcrypt
- REST API
- MongoDB Database Integration
- CRUD Operations
- Protected Routes
- Environment Variable Support

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- dotenv
- cors
- nodemon
